Road Distance Calculator

Powered by OpenRouteService

The extension uses a shared proxy by default — no setup needed.

To use your own key instead, get one free at openrouteservice.org .
How to use on a page:
Add these CSS classes to existing elements:
.road-distance-origin — start address
.road-distance-destination — end address
.road-distance-output — result goes here

Use data-rdc-group="id" for multiple pairs.
